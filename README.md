# Financial Fraud Detection Using Machine Learning and Power BI

## Project Overview
Built an end-to-end fraud detection system using Python, Random Forest, and Power BI using 6.3 million+ financial transactions.

## Key Results
- Precision: 100%
- Recall: 76%
- F1 Score: 86%

## Fraud Insights
Fraud concentrated mainly in:
- CASH_OUT
- TRANSFER

Fraud Rate:
0.13%

## Tools Used
- Python
- Scikit-learn
- SMOTE
- Power BI
